# BarelyReal Deskflow Fork

This repository is a public fork of [deskflow/deskflow](https://github.com/deskflow/deskflow), focused on keyboard, mouse, and clipboard sharing experiments that are relevant to [BarelyReal](https://github.com/greendexd/BarelyReal).

Deskflow is the upstream open-source project. This fork keeps upstream attribution and license terms intact, while tracking small usability, release, and integration ideas that can either help BarelyReal directly or be contributed back upstream when they are broadly useful.

## Current Focus

- Cross-platform keyboard and mouse sharing workflows for macOS and Windows.
- Safer clipboard and input-sharing behavior.
- Desktop setup and troubleshooting improvements.
- Small upstream contributions that improve the shared open-source ecosystem.
